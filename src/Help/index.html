QuSHAPE

QuSHAPE is a software suite for extracting quantitative nucleic acid reactivity information from capillary electrophoresis electropherograms.

To get started, please see the Quick Start Guide. This article describes QuShape algorithms in detail and describes the analysis, from raw data, of a comprehensive RNA structural probing experiment section.
